# markdown基本语法

1. 标题设置

    使用 #，一级标题为1个 #，2级标题为2个 #，以此类推，最多可设置6级标题。# 与标题文字之间必须有空格。

    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题
    
2. 文字书写

    - 正文书写：直接书写文字即可。如果需要在不空行的情况下换行需先敲至少2个空格再敲回车。

    - 字体： 倾斜、字体 加粗 以及字体 加粗+倾斜 的方法如下：

        *倾斜*
        **加粗**
        ***加粗倾斜***
    - 添加脚注：语法如下:

        需注释内容[^序号]
        ...文章内容...
        [^序号]: 注释
        其中，[^序号]: 注释 一般放在文末。实例：

        EGC[^1] patients usually have good prognosis.
        [^1]: EGC means early gastric cancer.


3. 列表

    * 无序列表：在列表每项前用 * 标记。* 与文字之间必须有一空格。


      * 第一项
      * 第二项
      * 第三项
      * 第四项
    * 有序列表：在列表每项前用数字加 . 表示。. 与列表文字之间必须有一空格。

      1. 第一项
      2. 第二项
      3. 第三项
      4. 第四项

4. 区块

    - 引用区块：在段落开头加 > 符号即可引用区块。> 符号与文字之间必须有空格。

        区块引用
        > 小白read.csv
        > Markdown 学习
    - 区块嵌套：一个 > 符号表示最外层，两个 > 表示第一层嵌套，以此类推。两个 > 之间需有一空格。

        > 最外层
        > > 第一层嵌套
        > > > 第二层嵌套
        > > > > 第三层嵌套 
    - 区块中使用列表：

        > 区块中使用列表
        > 无序列表
        > * 第一项
        > * 第二项
        > * 第三项
        > 有序列表
        > 1. 第一项
        > 2. 第二项
        > 3. 第三项

    - 列表中使用区块：在列表中使用区块时，需要在 > 前添加四个空格缩进。实例如下：

      * 第一项
          > 安装 VS Code。
          > 书写 markdown。
      * 第二项
          > 学习 git
          > 学习 R

5. Markdown 中显示代码

    - 句子中间出现的短代码片段：可以用一对 ` 包裹起来。例如：

        In `cin.get()`, the function `get()` is the member function of object `cin`.
        代码区块：可用一对 ``` 来包裹一段代码，形成代码区块，并指定一种语言（也可不指定语言）。例如：

        ``` C++
        #include <iostream>
        int main()
        {
            cout << "Hello world!" << endl;
            return 0;
        }
        ```
6. 链接

    - 链接使用方法主要包括如下三种：
        
        1. [链接名称](链接地址)
            例如：

            这篇文章主要参考了[菜鸟教程](https://www.runoob.com/markdown/md-link.html)
            直接使用链接地址：<链接地址>；例如：

        2. <https://www.runoob.com/markdown/md-link.html>
            变量设置链接，变量赋值在文档末尾进行。

        3. 这篇文章主要参考了[菜鸟教程][1]

            ...此处省略不知多少字...

            [1]: https://www.runoob.com/markdown/md-link.html

    - 通过 URL 插入：方法与插入链接相同，只需要在方括号前加上!。实例：

        ![pic](https://upload-images.jianshu.io/upload_images/21675413-82f44a6816582531.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240) 

        ![pic][1]

        ...文章内容...

        [1]:https://gitee.com/zhong_kai_zhao/xb/raw/master/images_firstmdnote/pic.png 
    - 插入本地图片：与通过 URL 插入相似，将括号内的内容替换为文件路径即可（本地路径貌似不能通过设置变量插入）。实例：

        括号里的路径可以为相对路径：
        [图片上传失败...(image-66386-1585641856556)] 

        也可以为绝对路径：
        ![startup])(D:\zzk\programming\git\learn_git\notes\images\VSCode_startup.png)

    - 本地路径貌似不能通过设置变量插入
        插入 base64 编码的图片：好处是图片就在文档里，不会丢。base64 编码极长，建议应用设置变量的方式插入。文末注释变量时注意 base64 编码前需要加上 data:image/png;base64,。实例：

        ![avatar][ao]
        ...base64 编码太长，全写上就传不上去了，以下是省略版...
        [ao]:data:image/png;base64,iVBOR...kJggg==

8. 表格

    - 表格的语法格式：

        |表头|表头|表头|
        |----|----|----|
        |单元格|单元格|单元格|
        |单元格|单元格|单元格|
        
        `|用于分隔同一行中的不同单元格，- 用于分隔表头和其他行。`
      

    - 设置表格对齐方式的方法：

        -: 设置内容和标题栏居右对齐。
        :- 设置内容和标题栏居左对齐。
        :-: 设置内容和标题栏居中对齐。
        实例如下：

        |表头|表头|表头|
        |----:|:----:|:----|
        |单元格|单元格|单元格|
        |单元格|单元格|单元格|
